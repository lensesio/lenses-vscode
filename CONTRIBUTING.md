# Contributing

Thank you for your interest in Lenses.io for VS Code!

## Reporting Issues

This is an **issues-only repository** for the Lenses.io VS Code extension. The source code is maintained in a private repository.

### Before Opening an Issue

1. **Search existing issues** to avoid duplicates
2. **Use the latest version** of the extension
3. **Check the [documentation](https://docs.lenses.io)** first

### How to Report a Bug

Please include:

- **VS Code Version**: Check `Help → About`
- **Extension Version**: Check Extensions panel
- **Lenses.io API Version**: If known
- **Operating System**: macOS, Windows, Linux
- **Steps to Reproduce**: Clear, numbered steps
- **Expected Behavior**: What should happen
- **Actual Behavior**: What actually happens
- **Screenshots**: If applicable
- **Logs**: From `View → Output → Lenses.io`

### Feature Requests

We welcome feature suggestions! Please:

- **Check existing issues** for similar requests
- **Describe the use case** clearly
- **Explain the benefit** to users
- **Provide examples** if possible

### Support

For general questions and support:

- **Documentation**: https://docs.lenses.io
- **Email**: info@lenses.io
- **Website**: https://lenses.io

## Code of Conduct

We expect all interactions to be:

- **Respectful** and professional
- **Constructive** and helpful
- **Focused** on the issue at hand

Thank you for helping us improve Lenses.io for VS Code! 🚀
